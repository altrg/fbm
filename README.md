Cowboy/SSL example
==================

Cowboy/SSL app example

Getting started
---------------

* git clone git://github.com/altrg/fbm.git
* make deps
* make run

Usage
-----

* App config file: fbm.cfg
* Log macroses: ?D()
* Adding apps: edit rebar.config, make deps
* Releases, upgrades, etc: see rebar docs
